## Base URL

Todas as requisições da API devem ser feitas para:

```
https://carteira.nevermissapps.com/v1
```

## Autenticação

Todas as requisições requerem autenticação via API Key no header:

```
Authorization: Bearer SUA_API_KEY
```

## Taxas

- **Pagamentos**: 6.99% sobre o valor
- **Saque Normal**: R$ 0,50 (24h)
- **Saque Turbo**: R$ 3,50 (instantâneo)

## Rate Limits

- **Geral**: 60 req/min
- **POST /v1/register**: 30s entre requisições
