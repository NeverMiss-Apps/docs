---

## Descrição

Retorna uma lista paginada de todas as suas transações (pagamentos e saques) com suporte a filtros por tipo e status.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token com sua API Key
</ParamField>

## Query Parameters (todos opcionais)

<ParamField query="limit" type="number" default={50}>
  Quantidade de resultados por página (máximo: 100)
</ParamField>

<ParamField query="offset" type="number" default={0}>
  Deslocamento para paginação
</ParamField>

<ParamField query="type" type="string">
  Filtrar por tipo: `payment` ou `withdrawal`
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status: `pending`, `approved`, `refunded`, `scheduled`, `processing`
</ParamField>

## Paginação

Para percorrer todas as transações:

```javascript
async function getAllTransactions() {
  const allTransactions = [];
  let offset = 0;
  const limit = 100;
  
  while (true) {
    const response = await fetch(
      `https://carteira.nevermissapps.com/v1/transactions?limit=${limit}&offset=${offset}`,
      { headers: { 'Authorization': `Bearer ${API_KEY}` } }
    );
    
    const data = await response.json();
    allTransactions.push(...data.transactions);
    
    if (!data.pagination.hasMore) break;
    offset += limit;
  }
  
  return allTransactions;
}
```
