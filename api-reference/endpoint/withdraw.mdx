## Descrição

Solicita um saque do seu saldo disponível para a chave PIX cadastrada. Existem dois tipos: **Normal** (R$ 0,50 - 24h) e **Turbo** (R$ 3,50 - instantâneo).

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token com sua API Key
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Deve ser `application/json`
</ParamField>

## Body

<ParamField body="turbo" type="boolean" default={false}>
  `true` para saque turbo (instantâneo), `false` ou omitir para saque normal (24h)
</ParamField>

<ResponseField name="success" type="boolean">
  Indica sucesso
</ResponseField>

      Valor líquido que receberá

## Comparação: Normal vs Turbo

| | Normal | Turbo |
|---|--------|-------|
| **Taxa** | R$ 0,50 | R$ 3,50 |
| **Tempo** | até 24h | Instantâneo |
| **Quando usar** | Sem urgência | Precisa urgente |
| **Economia** | Mais econômico | Mais rápido |

<Tip>
**Escolha o tipo certo**: Use saque normal para economia no dia a dia. Use saque turbo apenas quando realmente precisar do dinheiro instantaneamente.
</Tip>

## Erros Comuns

| Código | Causa | Solução |
|--------|-------|---------|
| 400 | Chave PIX não configurada | Configure via `/v1/register` |
| 400 | Saldo menor que R$ 5,00 | Aguarde mais pagamentos |
| 403 | Saque automático ativo | Desative via `/v1/register` |
| 409 | Já há saque pendente | Aguarde processamento |
