---

## Descrição

Consulta seu saldo atual disponível para saque e o valor de qualquer saque pendente.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token com sua API Key
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="balance" type="object">
  Informações do saldo
  
  <Expandable title="properties">
    <ResponseField name="available" type="number">
      Saldo disponível para saque em R$
    </ResponseField>
    
    <ResponseField name="pendingWithdrawal" type="number">
      Valor em saque pendente (se houver)
    </ResponseField>
    
    <ResponseField name="currency" type="string">
      Moeda: `BRL`
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://carteira.nevermiss.com/v1/balance \
  -H "Authorization: Bearer SUA_API_KEY"
```

```javascript Node.js
const response = await fetch('https://carteira.nevermiss.com/v1/balance', {
  headers: {
    'Authorization': 'Bearer SUA_API_KEY'
  }
});

const data = await response.json();
console.log(`Saldo disponível: R$ ${data.balance.available}`);
```

```python Python
import requests

response = requests.get(
    'https://carteira.nevermiss.com/v1/balance',
    headers={'Authorization': 'Bearer SUA_API_KEY'}
)

data = response.json()
print(f"Saldo disponível: R$ {data['balance']['available']}")
```

```php PHP
<?php
$ch = curl_init('https://carteira.nevermiss.com/v1/balance');

curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer SUA_API_KEY'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response);
curl_close($ch);

echo "Saldo disponível: R$ " . $data->balance->available;
?>
```

</RequestExample>

<ResponseExample>

```json Sucesso (200)
{
  "success": true,
  "balance": {
    "available": 150.75,
    "pendingWithdrawal": 0,
    "currency": "BRL"
  }

```

```json Com saque pendente (200)
{

  "balance": {
    "available": 150.75,
    "pendingWithdrawal": 100.00,
    "currency": "BRL"
  }
}
```

</ResponseExample>

## Observações

<Note>
**Saldo disponível**: Este valor já desconta qualquer valor congelado (MEDs - Medida de Exclusão de Débito) aplicado pelo sistema.
</Note>

<Info>
**Saque pendente**: Se houver um saque em processamento, o valor aparecerá em `pendingWithdrawal`. Este valor ainda não foi deduzido do seu saldo disponível.
</Info>

## Quando consultar o saldo

- Antes de solicitar um saque
- Após receber notificação de pagamento aprovado
- Para exibir saldo em dashboard/painel
- Para verificar se há saque em andamento
