## Configure em 3 passos

Comece a receber pagamentos PIX rapidamente seguindo estes passos simples.

### Passo 1: Obtenha sua API Key

Acesse o dashboard da NeverMiss Apps para gerar sua API Key:

<Card
  title="Gerar API Key"
  icon="key"
  href="https://nevermissapps.com/dashboard/configuracoes"
  horizontal
>
  Acesse as Configurações no Dashboard
</Card>

<Note>
**Dashboard completo**: No dashboard você também pode configurar visualmente todas as opções da sua conta, incluindo chave PIX, saque automático, dados da empresa (nome, logo, cor), webhook e callback URL.
</Note>

### Passo 2: Configure suas informações (Opcional)

<Tip>
**Recomendado**: Use o [Dashboard - Configurações](https://nevermissapps.com/dashboard/configuracoes) para configurar visualmente chave PIX, saque automático, dados da empresa (nome, logo, cor), webhook e callback URL.
</Tip>

Alternativamente, você pode configurar via API usando o endpoint `/v1/register`:

<CodeGroup>

```bash cURL
curl -X POST https://carteira.nevermissapps.com/v1/register \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "pixKey": "seu@email.com",
    "pixKeyType": "EMAIL",
    "companyName": "Minha Empresa",
    "companyLogo": "https://exemplo.com/logo.png",
    "companyColor": "#7C3AED",
    "webhookUrl": "https://exemplo.com/webhook",
    "callbackUrl": "https://exemplo.com"
  }'
```

```javascript Node.js
const response = await fetch('https://carteira.nevermissapps.com/v1/register', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer SUA_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    pixKey: 'seu@email.com',
    pixKeyType: 'EMAIL',
    companyName: 'Minha Empresa',
    companyLogo: 'https://exemplo.com/logo.png',
    companyColor: '#7C3AED',
    webhookUrl: 'https://exemplo.com/webhook',
    callbackUrl: 'https://exemplo.com'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://carteira.nevermissapps.com/v1/register',
    headers={
        'Authorization': 'Bearer SUA_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'pixKey': 'seu@email.com',
        'pixKeyType': 'EMAIL',
        'companyName': 'Minha Empresa',
        'companyLogo': 'https://exemplo.com/logo.png',
        'companyColor': '#7C3AED',
        'webhookUrl': 'https://exemplo.com/webhook',
        'callbackUrl': 'https://exemplo.com'
    }
)

print(response.json())
```

```php PHP
<?php
$ch = curl_init('https://carteira.nevermissapps.com/v1/register');

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer SUA_API_KEY',
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'pixKey' => 'seu@email.com',
    'pixKeyType' => 'EMAIL',
    'companyName' => 'Minha Empresa',
    'companyLogo' => 'https://exemplo.com/logo.png',
    'companyColor' => '#7C3AED',
    'webhookUrl' => 'https://exemplo.com/webhook',
    'callbackUrl' => 'https://exemplo.com'
]));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>
```

### Passo 3: Crie seu primeiro pagamento

Agora você está pronto para criar sua primeira cobrança PIX!

<CodeGroup>

```bash cURL
curl -X POST https://carteira.nevermissapps.com/v1/payment/create \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 49.90,
    "description": "Plano Premium - Mensal",
    "redirectUrl": "https://exemplo.com/sucesso"
  }'
```

```javascript Node.js
const response = await fetch('https://carteira.nevermissapps.com/v1/payment/create', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer SUA_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 49.90,
    description: 'Plano Premium - Mensal',
    redirectUrl: 'https://exemplo.com/sucesso'
  })
});

const payment = await response.json();
console.log(payment);
```

```python Python
import requests

response = requests.post(
    'https://carteira.nevermissapps.com/v1/payment/create',
    headers={
        'Authorization': 'Bearer SUA_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'amount': 49.90,
        'description': 'Plano Premium - Mensal',
        'redirectUrl': 'https://exemplo.com/sucesso'
    }
)

payment = response.json()
print(payment)
```

```php PHP
<?php
$ch = curl_init('https://carteira.nevermissapps.com/v1/payment/create');

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [

    'Content-Type: application/json'
]);

    'amount' => 49.90,
    'description' => 'Plano Premium - Mensal',
    'redirectUrl' => 'https://exemplo.com/sucesso'
]));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$payment = curl_exec($ch);
curl_close($ch);

echo $payment;
?>
```

## Dicas importantes

<Note>
**Taxa da plataforma**: 6.99% sobre cada transação
- Pagamento de R$ 100,00 > Você recebe R$ 93,01
- A taxa é deduzida automaticamente do valor
</Note>

<Tip>
**Use a página de pagamento pronta**:
Configure uma `callbackUrl` para receber links de pagamento prontos. Redirecione seus clientes para o `paymentLink` e deixe nossa página cuidar do resto!
</Tip>

<Warning>
**Rate Limits**:
- 60 requisições por minuto
- Atualização de registro: 30 segundos entre chamadas
</Warning>
