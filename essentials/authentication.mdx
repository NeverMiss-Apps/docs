## API Key

Todas as requisições para a API Carteira NeverMiss requerem autenticação via API Key. Sua chave deve ser incluída no header `Authorization` de cada requisição.

### Formato do Header

```
Authorization: Bearer SUA_API_KEY_AQUI
```

### Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET https://carteira.nevermissapps.com/v1/balance \
  -H "Authorization: Bearer SUA_API_KEY"
```

```javascript Node.js
const response = await fetch('https://carteira.nevermissapps.com/v1/balance', {
  headers: {
    'Authorization': 'Bearer SUA_API_KEY'
  }
});
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer SUA_API_KEY'
}

response = requests.get(
    'https://carteira.nevermissapps.com/v1/balance',
    headers=headers
)
```

```php PHP
<?php
$ch = curl_init('https://carteira.nevermissapps.com/v1/balance');

curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer SUA_API_KEY'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);
?>
```

</CodeGroup>

## Segurança

<Note>
**Mantenha sua API Key segura!**
- Nunca compartilhe sua chave em código público
- Não commite sua chave em repositórios Git
- Use variáveis de ambiente em produção
- Rotacione suas chaves periodicamente
</Note>

### Boas Práticas

<AccordionGroup>
  <Accordion title="Use variáveis de ambiente">
    Armazene sua API Key em variáveis de ambiente, nunca no código:
    
    ```javascript Node.js
    const API_KEY = process.env.NEVERMISS_API_KEY;
    ```
    
    ```python Python
    import os
    API_KEY = os.getenv('NEVERMISS_API_KEY')
    ```
    
    ```php PHP
    $apiKey = getenv('NEVERMISS_API_KEY');
    ```
  </Accordion>
  
  <Accordion title="Proteja suas requisições server-side">
    Sempre faça chamadas à API do lado do servidor, nunca exponha sua API Key no frontend:
    
    ✅ **Correto**: Chame a API do seu backend
    ```javascript
    // backend/api/payment.js
    app.post('/create-payment', async (req, res) => {
      const response = await fetch('https://carteira.nevermissapps.com/v1/payment/create', {
        headers: { 'Authorization': `Bearer ${process.env.NEVERMISS_API_KEY}` },
        // ...
      });
    });
    ```
    
    ❌ **Errado**: Nunca faça isso no frontend
    ```javascript
    // frontend/payment.js - NÃO FAÇA ISSO!
    const response = await fetch('https://carteira.nevermissapps.com/v1/payment/create', {
      headers: { 'Authorization': 'Bearer sua_chave_exposta' }
    });
    ```
  </Accordion>
  
  <Accordion title="Rotacione suas chaves regularmente">
    Como medida de segurança, considere rotacionar suas API Keys periodicamente, especialmente se:
    - Suspeitar de comprometimento
    - Um membro da equipe com acesso sair
    - Como parte de auditoria de segurança regular
  </Accordion>
</AccordionGroup>

## Erros de Autenticação

### 401 Unauthorized

Ocorre quando a API Key está ausente ou inválida:

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key"
}
```

**Soluções:**
- Verifique se o header `Authorization` está presente
- Confirme que o formato é `Bearer SUA_API_KEY`
- Verifique se a API Key está correta
- Certifique-se de que a chave não foi revogada
